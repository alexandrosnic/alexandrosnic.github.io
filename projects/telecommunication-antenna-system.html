Telecommunication Antenna System

Description

This project focused on stabilizing a telescopic telecommunication antenna under dynamic extensions, where vibration and transient oscillations can break satellite lock. I modeled the system dynamics in state-space form and designed a control architecture optimized for both precision and rapid settling.

Using an LQR-based control strategy, I tuned gains to minimize control effort while maintaining near-zero steady-state error and fast damping. The result was a stable pointing behavior with lower mechanical stress and improved tracking reliability.

Technologies Used

  • Matlab
  • Simulink
  • State-space modeling
  • LQR / PID control
  • Control optimization
