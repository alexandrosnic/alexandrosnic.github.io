Pedestrian - Vehicle Interaction Project

Description

This project, part of my internship at the Cyprus Institute, analyzed the interaction between pedestrians and drivers in Nicosia. The study involved the use of Tobii eye-tracking glasses to capture non-typical signals from drivers, which were then used to model pedestrian behavior in a 3D simulator for autonomous vehicles research.

Technologies Used

  • Matlab
  • Python
  • Tobii eye-tracking glasses
