Finite Element Analysis of Femur Bone

Description

This project involved a complete biomechanical FEA workflow for a human femur reconstructed from CT-derived geometry. I handled segmentation cleanup, mesh preparation, and material-property assignment to create a physically meaningful simulation model.

I evaluated multiple loading regimes (from static standing to high-impact scenarios) to analyze stress distribution and deformation behavior, with a strong focus on mesh quality and result stability.

Technologies Used

  • ANSYS (FEA)
  • Medical-image segmentation and mesh processing
  • Solid modeling / CAD preprocessing
  • Biomechanical load-case analysis
