Scroll Expander

Description

This project centered on designing and optimizing a scroll expander for an Organic Rankine Cycle (ORC) setup. I modeled mass/energy balances, chamber geometry interactions, and transient flow behavior to estimate performance under practical operating constraints.

By combining Simulink-based dynamic modeling with geometry refinement and FEA-informed checks, I improved the expander's volumetric efficiency and reduced loss mechanisms tied to leakage and pulse dynamics.

Technologies Used

  • Matlab / Simulink
  • Thermodynamic modeling (ORC)
  • ANSYS / geometry optimization
  • Control-oriented system analysis
