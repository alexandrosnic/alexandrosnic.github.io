Image-Based Computer Vision

Description

This project captures my frame-based computer-vision work in robotics, from traffic-sign and lane understanding to semantic scene interpretation. I developed and tuned pipelines that bridge raw camera feeds to actionable detections for navigation and manipulation.

I built robust preprocessing and training workflows to improve generalization under lighting and viewpoint shifts, and integrated object-level perception outputs into larger autonomy stacks for real-time decision support.

Technologies Used

  • Python / C++
  • OpenCV
  • YOLO-based detection
  • PyTorch / TensorFlow / Keras
  • Intel RealSense / Luxonis OAK-D
