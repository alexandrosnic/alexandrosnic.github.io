Event-Based Computer Vision

Description

This work was developed through event-based computer-vision assignments based on research-paper implementations, using ROS and C++ on neuromorphic camera data. The focus was asynchronous perception for scenarios where frame-based methods fail due to blur and temporal sparsity.

I implemented image reconstruction via leaky-integrator temporal filtering (time-surface state and kernel-convolved intensity), optical-flow estimation through contrast maximization of warped events, and simultaneous mosaicing/tracking for rotating event-camera setups.

For panoramic mapping, I used event-based gradients with integration via Poisson reconstruction to produce stable brightness panoramas from asynchronous event streams.

Technologies Used

  • ROS
  • C++
  • Event-based cameras (neuromorphic sensors)
  • OpenCV
  • Leaky-integrator temporal filtering
  • Optical flow via warped-event contrast maximization
  • Mosaicing and Poisson-based image reconstruction
