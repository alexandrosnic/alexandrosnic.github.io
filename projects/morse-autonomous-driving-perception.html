Object Detection for Autonomous Driving in MORSE

Perception benchmarking and robustness tuning in simulation

Description

This project was part of a 5-person autonomous-driving system in the MORSE simulator, covering SLAM, perception, prediction, planning, navigation, and decision-making.

I was responsible for the perception module, where I implemented YOLO in ROS for object detection, dynamic semantic mapping, and generation of the perceived dynamic map consumed by downstream planning/decision layers.

Capabilities Demonstrated

  • Implemented ROS-based perception with YOLO-driven object detection in a full autonomous-driving stack.
  • Built dynamic semantic maps from live perception outputs for behavior and planning modules.
  • Validated autonomous behaviors such as free-parking-space recognition and parking attempts in simulation.
  • Integrated perception outputs across multi-module autonomy workflows under realistic simulator dynamics.

Technologies Used

  • Python
  • ROS
  • OpenCV
  • YOLO
  • MORSE Simulator
  • Dynamic semantic mapping
