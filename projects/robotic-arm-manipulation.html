Robotic Arm Manipulation

Description

The goal of this project was to develop a robotic arm framework for executing pick-and-place tasks. This framework integrated computer vision (CV) and natural language processing (NLP) modules into a ROS2-based environment. I used Webots for simulation, MoveIt2 for motion planning, and customized URDF/proto files for the OnRobot VGC10 gripper.

Technologies Used

  • ROS2, SMACH
  • Webots
  • MoveIt2
  • Docker
  • SolidWorks (CAD)
  • Custom ROS2 drivers
