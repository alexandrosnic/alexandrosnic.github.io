ROS1 to ROS2 Migration

Description

Modernizing robotics infrastructure is more than just rewriting code; it’s about ensuring reliability across generations of software. I’ve specialized in migrating legacy ROS1 packages to ROS2. This involved not only porting the communication logic but also refactoring ROS and URDF files to support the new ROS2 control paradigms, ensuring that high-precision industrial tools remain functional in modern, secure, and multi-threaded environments.

Key Achievement

Successfully migrated ROS1 drivers and description such as for the VGC10 robotic gripper to ROS2, enabling real-time suction control in a ROS2-only industrial pick-and-place framework.
