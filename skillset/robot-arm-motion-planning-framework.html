Robot Motion Planning & Manipulation Frameworks

Description

My focus in manipulation is the framework integration of the arm into the wider robotic stack. I use MoveIt to handle complex trajectory planning and kinematic control. I developed an API-driven architecture where high-level commands (e.g., 'pick object_ID') are translated into precise motion plans. While I can integrate computer vision and NLP/LLM modules into these stacks, my core work lies in managing the state machines (or behavior trees) and trajectory execution that make the physical interaction possible.

Key Achievement

Engineered a mission planner for the TIAGo and UR3e robots using MoveIt and finite state machines (SMACH), enabling them to execute complex pick-and-place sequences in both simulated and real environments.

Tools

  • ROS 1/2
  • C++/Python
  • MoveIt
  • Finite State Machines (SMACH, FlexBe)/Behavior Trees (BehaviorTree.CPP)
